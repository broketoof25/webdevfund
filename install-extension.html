How to install and use VS Code's Live Server Extension

  • Installation Instructions
  • Usage Instructions
  • Video

Installation Instructions

  1. In Visual Studio Code, click the Extension icon in the left column toolbar.
  2. In the search bar that appears, type Live Server.
  3. Select the Live Server extension. It should be the top choice
  4. In the page for Live Server that appears, click the green Install button.

Usage Instructions

  • Visual Studio Code must be open to a folder to use Live Folder.
  • In VS Code's Explorer, right click (window) or Control + click (macos) on an HTML file. From the context menu that appears, select Open with Live Server.
